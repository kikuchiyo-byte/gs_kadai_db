# ①課題名
会員登録アプリ

## ②課題内容（どんな作品か）
- PHPMailer + SMTPを利用したメール認証を行い、DBにデータを登録して管理する
- マイページでユーザー名とプロフィール画像を編集できる

## ④アプリのログイン用IDまたはPassword（ある場合）
- メール認証で誰でもログインできます

## ⑤工夫した点・こだわった点
- 公開サービスの水準向上を目指して、PHPMailer + SMTPを使用してみました。
- AIの提案を受け取って、検証したり、エラーが出たときの相談など、AIの使い分けができてきました。
- 部品を作ってもらって、組み合わせるほうが構造の理解も深まって組み合わせや修正が容易。しかし、修正アップグレードのたびにエラーが発生しやすいのが難点。自分の理解、知識不足でもあると思います。

## ⑥難しかった点・次回トライしたいこと（又は機能）
- 実際に提供されている公開サービスと比較しながら、足りないものを想像して調べるを繰り返してブラッシュアップしていきたい。
- 次はサービスのメイン部分にも着手したい
